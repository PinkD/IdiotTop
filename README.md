# 智障网刷题工具


## 软件介绍

- 由 [SuperMartin](http://supermartin.cn/) 开发，由 [PinkD](https://blog.pinkd.moe/) 重构。
- 采用题库刷题，自带题库可能不是最新版，请刷完后在 [至善网](http://www.attop.com/) 中手动提交正确答案，然后再提取题库，即可制作最新题库。
- 本程序开源，程序源代码可以在[PinkD的のgithub](https://github.com/PinkD)上找到
